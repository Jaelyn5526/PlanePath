PlanePath
=========

飞机沿着轨迹飞行，并且可以变速。<br>

1.飞机沿着手指划过的轨迹移动 <br>
2.移动的过程中，可以通过顶部3个按钮修改飞机的移动速度<br>
3.飞机移动的速度与手指划过屏幕的速度相关<br>
